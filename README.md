# My Reads App

My Reads is a simple book management application created built with React.

You can add the books that you want to read, currently reading or have read.

## Setting up

Clone or download the repository.

Run:
```
> npm install
> npm start
```

## Built With

* React
* JavaScript
* CSS
* HTML
